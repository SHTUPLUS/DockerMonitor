## 我申请的临时权限到期了之后所有文件是否删除?
* 如果本来就有集群使用权限，只是因为需要增加计算资源的情况，我们是会删除临时申请增加的节点权限，文件还是会保留。
* 如果原本就没有集群使用权限,比如因为课程需要，需要临时开通账号以及节点权限，使用到期后，我们先删除节点权限，此时admin权限还在，数据也可以访问. 然后我们会给额外一周的时间用来迁移数据，一周之后会删除admin权限以及账户资料。
